**MOLC Workflow**

This module provides a **database-driven workflow** for **MOLC (Molecular Cohorts)** using cloud/DuckDB data, including **signature scoring / TME deconvolution** and **downstream analysis** (visualization, group comparison, and correlation).

**🔄 Recommended Order**
1. Data Selection
2. TME Cluster
3. Visualization / Group Comparison / Correlation

**📌 Usage Notes**
- **Complete Part 1 first** (**Data Selection → TME Cluster**).
- Downstream tabs use the prepared cohort data and clustering results from Part 1.
- Most modules support **parameter tuning** and **plot size adjustment** for visualization and export.
